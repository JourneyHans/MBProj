# 发布说明

## [v0.1.0-alpha] - 2026-03-13

### 🎮 功能特性
- **核心玩法**: 完整的扫雷实现
  - 左键点击揭开方块
  - 右键点击标记地雷
  - 首次点击保护（第一次点击后生成地雷）
  - 自动揭开空白方块（泛洪填充）
  - 胜利/失败判定

- **渲染**: 基于Canvas的网格渲染
  - 彩色数字（1-8）
  - 不同方块状态的视觉反馈
  - 流畅的60 FPS渲染

- **用户界面**:
  - 主菜单
  - 抬头显示器（地雷计数器、计时器）
  - 胜利/游戏结束对话框
  - 简洁的深色主题界面

### 🏗️ 技术架构
- 使用现代JavaScript的ES6模块
- 事件驱动架构（EventBus）
- 状态管理系统
- 基于requestAnimationFrame的游戏循环
- 模块化、可维护的代码结构

### 🐛 已知问题
- 尚未实现卡牌系统（计划于v0.2.0）
- 尚未实现Roguelike元素（计划于v0.3.0）
- 基础视觉效果，无美术资源
- 无音效
- 未针对移动端触摸操作优化

### 🔧 浏览器支持
- Chrome/Edge: ✅ 完整测试
- Firefox: ✅ 完整测试
- Safari: ⚠️ 基础测试
- 移动浏览器: ⚠️ 有限支持

### 📊 性能指标
- 首次加载: ~1.5秒
- 运行时帧率: 60 FPS
- 打包大小: ~15KB（未压缩）

### 🎯 下一版本 (v0.2.0-alpha)
- 卡牌系统实现
- 卡组构建机制
- 基础卡牌效果
- 能量系统
