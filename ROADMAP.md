## 💡 下次开发从这里开始

### 如何继续开发

1. **拉取最新代码**
   ```bash
   git pull origin main
   ```

2. **创建功能分支**
   ```bash
   git checkout -b feature/card-system
   ```

3. **开始开发**
   - 从 `js/cards/Card.js` 开始
   - 参考本文档的 v0.2.0-alpha 功能列表
   - 遵循项目的代码规范

4. **测试和提交**
   ```bash
   # 本地测试
   python -m http.server 8080

   # 提交代码
   git add .
   git commit -m "feat(cards): implement card class"
   git push origin feature/card-system
   ```

5. **创建 Pull Request**
   - 在 GitHub 上创建 PR
   - 请求合并到 main 分支
